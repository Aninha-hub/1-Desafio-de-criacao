# 1-Desafio-de-criação
Criar template de newsletter para divulgação interna de novos produtos

💡 Desafio Criativo: Construindo um Prompt
Bem-vindo ao Desafio Criativo.
Nesta atividade, você seguirá algumas etapas simples que, ao final, irão ajudar a construir um prompt completo e bem estruturado.

💡 Sobre o Exercício
Um bom prompt nasce de intenção clara, contexto e instruções específicas. Durante o desafio, você irá organizar suas ideias e transformá-las em comandos claros.

🧩 Construção Passo a Passo
Cada etapa adicionará uma nova peça ao seu prompt final. Ao concluir todas, você terá um prompt capaz de orientar uma IA a produzir exatamente o que você imaginou.

🚀 Como Resolver
Siga cada passo com atenção, registre suas respostas e, ao final, una todos os elementos para formar seu prompt final.

Aproveite o processo criativo!

## Etapa 1 — 📩 O pedido inesperado da liderança (definindo o problema)
Você trabalha no time de Comunicação Interna de uma empresa de tecnologia. Numa manhã de segunda-feira, chega uma mensagem direta da liderança:

“Toda semana tem produto novo saindo, mas a galera só descobre tarde — ou descobre errado. Precisamos de uma newsletter interna clara, rápida e padronizada para divulgar os novos produtos. E precisamos disso de um jeito que qualquer pessoa consiga gerar sem sofrer.”

O problema: hoje, cada área escreve do seu jeito, faltam informações essenciais (o “por quê”, o “pra quem”, o “como usar”), e a newsletter vira uma colcha de retalhos.

Seu desafio será criar um template de prompt que gere uma newsletter interna consistente, bem organizada e pronta para publicar, a partir de informações básicas.

🎯 Objetivo da etapa
Antes de escrever o prompt, defina o que essa newsletter precisa sempre entregar, independentemente de quem escreve:

✅ Liste os blocos fixos que a newsletter deve ter (ex.: 🧠 resumo, 🚀 novidades, 👥 público-alvo, 🔗 links, 📣 CTA etc.)
✅ Para cada bloco, escreva 1 frase dizendo o objetivo daquele bloco (o que ele precisa comunicar)

## Etapa 2 — 🧩 Transformando “seções” em campos de entrada (o que o prompt vai pedir)
A liderança aprova a ideia da newsletter padronizada, mas acrescenta uma condição:

“Tem que funcionar mesmo quando a pessoa só tiver um punhado de informações. O prompt precisa puxar os dados certos e não deixar passar nada importante.”

Agora você precisa garantir que o seu template de prompt não dependa de inspiração — ele depende de inputs.

🧠 O que você vai fazer nesta etapa
Você vai transformar cada seção fixa da newsletter em perguntas/campos que a pessoa preenche antes de gerar o texto.

Pense assim: se a IA só pode escrever com base no que você fornece, quais campos são obrigatórios para não sair uma newsletter vaga?

✅ Para cada seção, defina:
Campos obrigatórios (o mínimo pra funcionar)

Campos opcionais (quando tiver mais detalhe, melhora a qualidade)

Restrições úteis (ex.: limite de caracteres, tom, evitar siglas sem explicar)

🎯 Objetivo da etapa
Criar um formulário de entrada (em texto) que o prompt vai usar, com:

Campos obrigatórios e opcionais

Regras rápidas de preenchimento (ex.: “se não souber, escreva ‘N/A’”)

## Etapa 3 — 🏗️ O esqueleto do prompt (papel, objetivo e formato)
Com as seções e os campos definidos, você já tem o “conteúdo bruto”. Mas ainda falta o mais importante: dar direção pra IA.

A liderança quer previsibilidade:

“Se eu rodar esse prompt toda semana, a newsletter precisa sair com o mesmo padrão: mesma estrutura, mesma clareza, mesmo tipo de chamada.”

Então agora você vai montar o template base do prompt — ainda sem refinamentos — só o núcleo que garante consistência.

🧱 O que você vai construir nesta etapa
Um prompt com 3 partes fixas:

🎭 Papel (Role) Definir quem é a IA (ex.: redator(a) de comunicação interna, focado em clareza e adoção).

🎯 Objetivo (Goal) Gerar uma newsletter interna de novos produtos a partir dos campos de entrada.

🧾 Formato de saída (Output format) Especificar a estrutura final: títulos, seções, bullets, CTAs, links, e um bloco “Resumo executivo”.

✅ Regras essenciais para incluir já no esqueleto:

Tom: interno, claro, direto, motivador (sem marketing exagerado)
Sempre explicar siglas na primeira vez
Se faltar informação, não inventar: sinalizar como “⚠️ Informação pendente”
Manter o texto escaneável (títulos curtos + bullets)
🎯 Objetivo da etapa
Ter um template de prompt em versão 1.0, com:

Papel + objetivo + regras
Estrutura fixa da newsletter (as seções)
Um bloco final para a pessoa colar os campos preenchidos

## Etapa 4 — ✅ Prompt final + 🧪 exemplo preenchido (pronto para copiar e usar)
Chegou a hora de entregar o que a liderança realmente quer: um template de prompt replicável + um exemplo realista pra qualquer pessoa do time entender como usar.

A ideia é simples:

1. a pessoa cola o template,
2. preenche os campos,
3. recebe uma newsletter pronta e padronizada.

🧠 Regras

# 📄 Entrega do Prompt Final

Após concluir o Desafio Criativo, você deverá entregar seu prompt final por meio de um documento no Google Docs.

✍️ Passo 1: Criar o documento

1. Acesse: https://docs.google.com

2. Clique em "Em branco" para criar um novo documento.

3. Dê um título ao documento, por exemplo:

Desafio Criativo – Prompt Final

4. Cole no documento o prompt que você construiu durante a atividade.

🔗 Passo 2: Tornar o documento público

Para que possamos acessar seu trabalho, é necessário compartilhar o documento de forma pública.

1. Clique no botão "Compartilhar" no canto superior direito.

2. Na seção "Acesso geral", clique no menu que normalmente aparece como "Restrito".

3. Selecione a opção "Qualquer pessoa com o link".

4. Defina a permissão como "Leitor".

📤 Passo 3: Enviar o link

1. Clique em "Copiar link".

2. Cole esse link no local indicado para envio da atividade.

✅ Importante

Antes de enviar, verifique se:

* O documento contém seu prompt final completo

* O link está configurado como "Qualquer pessoa com o link pode visualizar"

Assim garantimos que seu trabalho possa ser acessado corretamente. 🚀
